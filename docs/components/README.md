# Components

> Source: `src/components/`

Lizard UI components are split into two layers:

| Layer | Path | Contents |
|---|---|---|
| **UI primitives** | `src/components/ui/` | `Button`, `Card`, `Badge`, `BackgroundPattern`, `Header`, `Footer`, `Sidebar`, `VerticalMenu`, `GitHubFooter` |

All components are exported from the library root:

```ts
import {
  Button,
  Card, CardHeader, CardTitle, CardDescription, CardContent, CardFooter,
  Badge,
  BackgroundPattern,
  Header,
  Footer,
  Sidebar,
  VerticalMenu,
  GitHubFooter,
} from 'lizard-ui';
```

See [components/ui](./ui/README.md) for per-component API reference.
